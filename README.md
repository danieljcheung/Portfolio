# Daniel Cheung Portfolio

Personal portfolio website showcasing projects in cybersecurity and AI.

## Tech Stack

- **HTML/CSS** - Tailwind CSS via CDN
- **JavaScript** - Vanilla JS + Anime.js for animations
- **Fonts** - Space Grotesk, IBM Plex Serif, JetBrains Mono

## Features

- Dark/light mode with animated wash transition
- Custom terminal cursor (`>`) with click effects
- Scroll-triggered animations
- Responsive mobile menu
- Contact modal

## Local Development

Open `index.html` in a browser. No build step required.
